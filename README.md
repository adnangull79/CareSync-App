CareSync is a healthcare app built using Kotlin and Jetpack Compose.  
It helps users book doctor appointments, store medical records, and chat with an AI health assistant.  

Features:
- Doctor appointment booking  
- AI health chatbot  
- Personal health record storage  
- Doctor private notes system  
- Basic health calculators  

Tech Stack:
- Kotlin  
- Jetpack Compose  
- Firebase Firestore  
- Firebase Authentication  
- MVVM Architecture
